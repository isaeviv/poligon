Videosex — югославская синти-поп-группа из Любляны. Название для коллектива придумал Деян Кнез, известный своим участием в проекте Laibach.

В 1983 году группа дебютировала с синглом «Moja Mama». Web-приложение – это приложение работающее на архитектуре клиент – сервер. Согласно этой архитектуре, приложение располагается на удаленном сервере, а пользователь взаимодействует с интерфейсом, отправляя запросы на сервер через браузер, запущенный на своем компьютере. Для работы с приложением нет необходимости устанавливать какое-либо дополнительное ПО на стороне клиента, за исключением браузера. Достаточно лишь перейти на URL адрес приложения для того чтобы увидеть его интерфейс. Работа клиента не зависит от ОС на компьютере пользователя, поэтому нет нужды разрабатывать отдельные версии для Linux, Widows, Mac OS или BSD. Чаще всего разработка серверной части web-приложения ведется на таких языка как PHP, Python, C#, Java, Ruby и т.д. Для реализации клиентской части приложения принято использовать HTML, XHTML, CSS, JavaScript, CoffeScript.
